# ``Digest/Base64``

## Overview

Base64 exists to provide a namespace. It contains static methods for Base64 encoding and decoding
and for conversion between byte arrays and hexadecimal strings. There is no Base64 instances. 

## Topics

### Methods

- ``encode(_:_:)``
- ``decode(_:)``
- ``pemEncode(_:_:_:)``
- ``pemDecode(_:_:)``
- ``hex2bytes(_:)``
- ``bytes2hex(_:_:)``
